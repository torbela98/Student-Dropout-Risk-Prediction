# Student-Dropout-Risk-Prediction

The repository contains my capstone project, Predicting Student Dropout Risk Using Machine Learning. The project presents an end-to-end machine learning workflow for predicting student dropout risk using the UCI Predict Students Dropout and Academic Success dataset. The objective is to develop an accurate and interpretable predictive model that can assist higher education institutions in identifying students who may be at risk of dropping out, enabling timely and evidence-based interventions to improve student retention and academic success.

The project was implemented in Python using Google Colab and followed a structured data science workflow consisting of data preparation, exploratory data analysis (EDA), feature engineering, feature selection using SelectKBest, dimensionality reduction through Principal Component Analysis (PCA), model development, model evaluation, hyperparameter tuning using GridSearchCV, explainability using SHAP (SHapley Additive exPlanations), and fairness assessment. Four supervised machine learning algorithms—Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting—were trained and evaluated using Accuracy, Precision, Recall, F1-score, Area Under the Receiver Operating Characteristic Curve (AUC), confusion matrices, and five-fold cross-validation. Based on the evaluation results, the optimized Random Forest model was selected as the final predictive model, achieving an AUC of approximately 0.972.

The repository includes the Google Colab notebook, the dataset obtained from the UCI Machine Learning Repository, project documentation, the written capstone report, and two presentation decks prepared for different audiences: a Technical Presentation for peers, and a Business Presentation for senior executives that highlights the project's institutional value and decision-support applications. Together, these materials document the complete lifecycle of the project, providing a comprehensive reference for applying machine learning to educational predictive analytics.
